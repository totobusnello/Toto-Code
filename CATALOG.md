```bash

/sc:design

# 2. Implementar
autopilot: implement the designed architecture

# 3. Testar
/sc:test

# 4. Documentar
/sc:document
```

### 2. Bug Fix Rapido

```bash
# Usar modo economico
eco: fix the bug in auth module
```

### 3. Feature Complexa

```bash
# Usar Ralph para persistencia
ralph: implement payment integration with Stripe
```

### 4. Code Review

```bash
# Review automatico
/omc:code-review
```

### 5. Security Audit

```bash

# Agent: security-auditor
```
