## Como Usar

Para adicionar um repositório externo ao seu projeto:

```bash
# Clone o repositório desejado
git clone https://github.com/user/repo.git external/repo-name

# Ou adicione como submódulo
git submodule add https://github.com/user/repo.git external/repo-name
```

## Nota

Os symlinks anteriores foram removidos pois apontavam para diretórios inexistentes.
