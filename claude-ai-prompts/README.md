# Skills para Claude.ai (Web)

> 60 skills exportadas para usar no Claude.ai (interface web)

## Como Usar

### Opcao 1: Copiar no Inicio do Chat

1. Abra o arquivo da skill desejada
2. Copie todo o conteudo
3. Cole no inicio de uma nova conversa no claude.ai
4. Faca sua pergunta/tarefa

**Exemplo:**
```
[Cole o conteudo de 01-copywriting.md aqui]

---

Agora, escreva copy para minha landing page de SaaS de gestao de projetos.
```

### Opcao 2: Usar Projects (Claude Pro)

1. Acesse claude.ai/projects
2. Crie um novo projeto
3. Adicione o arquivo .md como documento
4. Use o chat do projeto

### Opcao 3: Custom Instructions (System Prompt)

Se tiver acesso a API ou custom instructions:
1. Use o conteudo da skill como system prompt
2. Envie suas mensagens normalmente
