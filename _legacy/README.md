# Legacy

> Skills e agents legados que ainda funcionam mas usam padroes antigos.

## Quando Usar

- Compatibilidade com sistemas antigos
- Migracao gradual para novos padroes
- Referencia para entender evolucao

## Migracao

Para migrar de um item legacy para a versao atual:

1. Identifique o equivalente atual em `skills/` ou `agents/`
2. Compare as funcionalidades
3. Atualize suas referencias
4. Teste completamente antes de remover o legacy

## Estrutura

```
_legacy/
├── skills/      # Skills legados
├── agents/      # Agents legados
└── README.md
```
