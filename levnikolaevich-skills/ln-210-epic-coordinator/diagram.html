üîë Key Features v7.0.0

  • HTML Research: Extract domains from navigation, forms, page titles (complements documentation)
  • Epic 0 Reserved: Infrastructure Epic = Epic 0 index, business domains = Epic 1-N
  • Batch Preview: Show ALL Epics before creation, single confirmation (not sequential loop)
  • Auto-extraction Q1-Q5: Goal, Scope In/Out, Success Criteria, Risks from docs + HTML
  • CREATE/REPLAN Modes: Decompose-First Pattern - builds IDEAL plan, compares with existing, determines operations
  • Business Epic Grouping: 1 Epic = complete business process OR HTML screen group (5-10 Stories)

üìê Phase Structure

  • Phase 1: Discovery & Research (Team ID, docs, HTML, Infrastructure decision)
  • Phase 2: Scope Analysis (identify domains, build Epic list, user confirms structure)
  • Phase 3: Check Existing Epics (determine CREATE vs REPLAN mode)
  • Phase 4: Epic Preparation (CREATE only - auto-extract, generate docs)
  • Phase 5a: Epic Creation (CREATE only - batch preview, confirm, create ALL)
  • Phase 5b: Epic Replan (REPLAN only - compare, show operations, execute)
